# Sentinel House Press website

A responsive static HTML website prepared for GitHub Pages and the custom domain `sentinelhousepress.com`.

## Creative direction

The website uses a Sentinel Editorial Modernism system:

- Bone paper, ivory, ink black, deep crimson, oxblood, muted olive, blue-black, and burnt umber
- Large literary typography with strong changes in scale
- Page-specific visual concepts rather than one repeated layout
- Real founding-book imagery and custom cover-detail crops
- Editorial proof marks, book layers, running heads, folios, archive captions, and a colophon
- Fast, restrained page transitions and scroll reveals
- A fixed publishing-spine progress indicator and subtle artifact hover labels
- Page-specific color atmospheres and restrained perspective movement
- Responsive and reduced-motion behavior

## Business strategy reflected in the site

- Protect, Produce, Prosper operating lens
- Author-funded professional services as the default model
- Clear scope, deliverables, timelines, revision limits, responsibilities, exclusions, and milestone payments
- Author ownership of copyright, publishing accounts, files, and decisions
- Strong-fit and poor-fit author qualification
- A founder-led operating model that protects quality and workload
- Professional guidance, author control, and books built to last
- Four clearly framed engagement paths: assess, develop, publish, and extend
- Documented handoff standards, author-control promises, and honest proof boundaries

## Before launch

1. Connect the project assessment form in `contact.html` to Formspree, Basin, or another secure form endpoint.
2. Add a verified company email and update the privacy page.
3. Add analytics and Search Console verification after choosing the final accounts.
4. Have the privacy notice, service agreement, proposal terms, and client contract reviewed by qualified counsel.
5. Confirm DNS and GitHub Pages custom-domain settings.
6. Test every page on Safari, Chrome, Firefox, iPhone, and Android after deployment.

## GitHub Pages

Upload the contents of this folder to the root of the GitHub repository. The `CNAME` file contains `sentinelhousepress.com`. In repository settings, select the deployment branch and confirm the custom domain.

## Plymouth location page

`plymouth.html` presents Sentinel House Press as proudly based in Plymouth, Massachusetts, known as America’s Hometown. It includes original vintage-style landmark artwork and contextual language recognizing Plymouth’s layered Wampanoag, English, civic, and contemporary histories.


## V10 responsive architecture

This edition consolidates the live layout into a permanent `sentinel-site` system. Historical body classes from V4 through V7 were removed. The final CSS adds standard media frames, a stable grid-based hero, a single breakpoint hierarchy at 1199px, 900px, 640px, and 380px, protected 3:5 book-cover proportions, simplified mobile artwork, accessible menu focus handling, Escape-to-close behavior, and a delayed mobile manuscript CTA.

Test before deployment at 320, 375, 390, 430, 768, 1024, 1280, and 1600 pixels, plus browser zoom at 200 percent.


## V10 typography system

This build uses a three-family system:

- Bodoni Moda Variable for display typography and conceptual headlines
- Newsreader Variable for body copy, essays, case studies, and literary italics
- Manrope Variable for navigation, buttons, forms, facts, prices, and metadata

The final CSS layer adds a fluid type scale, improved reading measures, sentence-case buttons, tabular numerals for metrics, old-style numerals for prose, page-specific typographic treatments, and responsive typography down to 320px. Older editorial version classes were removed from every page body.
