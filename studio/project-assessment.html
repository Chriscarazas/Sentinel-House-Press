A useful first note

Give us the shape of the problem.

Your message does not need to be polished. It only needs enough context for Sentinel House Studio to understand the project and recommend a sensible next step.

  • Project type
  • Intended audience
  • Existing source material
  • What is currently unclear or ineffective
  • Desired outcome and deadline

Please do not attach confidential reports, beneficiary information, donor data, or sensitive source files in the first message. Secure transfer can be arranged after fit is confirmed.
